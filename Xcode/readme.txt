RoboMonitor

iOS App for monitoring FIDO autonomous robots.

Principal features:

1. Communicates with the robot via Bluetooth Low Energy, or UDP.

2. Configured by messages from the robot.

2. Monitors and controls system state - eg power/activity level.

3. Monitors and controls subsystem status, performance and settings.

4. Displays Log Messages from errors and other events.

5. Limited remote control capability.
